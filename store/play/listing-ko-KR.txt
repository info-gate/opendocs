### 앱 이름 (App name) — 30자 제한
HWP·PDF 뷰어 OpenDocs - 무료

### 짧은 설명 (Short description) — 80자 제한
HWP, PDF, Word, Excel, 한글 문서 모두 무료 — 회원가입 X, 광고는 적게, 즉시 열림

### 자세한 설명 (Full description) — 4000자 제한 / 해시태그 0개
모든 문서를 한 번에 열어보세요. HWP·PDF·DOCX·XLSX·PPTX·EPUB·이미지 — OpenDocs 하나면 충분합니다.

회원가입 없이, 구독 없이, 인앱 결제 없이.
가벼운 무료 뷰어가 필요했다면 OpenDocs 입니다.

[지원 형식]
• HWP / 한글 (Hancom) — 한국 사용자를 위한 네이티브 지원
• PDF — 빠른 페이지 넘김 + 핀치 줌
• DOCX (Word) / XLSX (Excel) / PPTX (PowerPoint)
• EPUB — 전자책 리더
• TXT — 일반 텍스트
• 이미지 — JPG, PNG, WEBP, GIF

[왜 OpenDocs 인가]
✓ HWP 무료 뷰어 — 한컴 오피스 유료 구독 없이 한글 문서 열기
✓ PDF 무료 리더 — Adobe 로그인 없이 즉시 사용
✓ 회원가입 없는 문서 뷰어 — 권한 최소화, 데이터 외부 전송 X
✓ 가벼움 — 폐기된 노후 기기에서도 빠르게 실행
✓ 광고 위치 분리 — 문서 보는 화면에는 광고 X (방해 없음)

[주요 기능]
• 빠른 실행 — 무거운 오피스 앱 대신 즉시 열림
• 깔끔한 뷰어 — 페이지 넘김, 확대/축소, 다크 모드
• 최근 파일 — 자주 보는 문서 한 번에 정리
• 즐겨찾기 — 중요한 파일 별도 표시
• 다국어 UI — 한국어, 영어, 일본어
• 프라이버시 우선 — 모든 처리는 기기 안에서, 서버 업로드 없음

[누구에게 좋나요]
• 거래처에서 받은 HWP 문서를 외출 중 빠르게 확인해야 하는 직장인
• 학교 과제 한글 파일을 핸드폰에서 열어야 하는 학생
• PDF, Word, Excel을 모바일에서 자주 보는 모든 사용자
• 무거운 오피스 앱 대신 가벼운 뷰어가 필요한 사용자
• 광고 많은 다른 뷰어 앱이 부담스러웠던 사용자

[다른 뷰어 앱과 비교]
• 한컴 오피스: HWP 전용, 유료 구독 또는 일부 기능 제한, 로그인 필요
• Adobe Reader: PDF 전용, HWP 미지원, 일부 기능 유료, 로그인 권장
• WPS / Microsoft 365: HWP 미지원, 광고/구독 부담
• OpenDocs: HWP + PDF + Word + Excel + EPUB + 이미지 모두 무료, 회원가입 X

[자주 묻는 질문]

Q. HWP 파일이 정말 깨지지 않고 열리나요?
A. 네. 한글(HWP) 표준 포맷을 네이티브 지원합니다. 텍스트, 표, 이미지가 모두 표시됩니다.

Q. 문서를 편집할 수 있나요?
A. 현재 버전은 뷰어(읽기 전용)입니다. 빠른 확인과 공유에 최적화되어 있습니다.

Q. 인터넷 없이 사용 가능한가요?
A. 네. 모든 처리는 기기 안에서 이루어집니다. 서버 업로드 없이 오프라인에서 작동합니다.

Q. 정말 무료인가요? 숨겨진 결제 있나요?
A. 완전 무료입니다. 인앱 결제 없음, 구독 없음, 프리미엄 잠금 기능 없음. 광고로 운영됩니다.

Q. 한국 외 사용자에게도 유용한가요?
A. 네. 영어, 일본어 UI 지원. PDF, DOCX, XLSX, EPUB, 이미지 등 글로벌 포맷도 모두 지원합니다.

OpenDocs 는 여러분의 시간을 아껴주기 위해 만들었습니다.
지금 설치하고 모든 문서를 한 번에 열어보세요.
